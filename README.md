# Almila Masal Oyunu

Almila için Scratch ile geliştirilen, kişiselleştirilebilir masal/operasyon oyunu.

## Oyun fikri

Oyuncu Almila karakterini yönetir. Bölümler ilerledikçe sahne değişir:

1. Masal Ormanı
2. Uzay Yolculuğu
3. Kale Kapısı
4. Prenses Kurtarma Finali

Amaç yıldızları toplamak, engellerden kaçmak ve geçitten geçerek yeni seviyeye ulaşmaktır.

## Kontroller

- Sağ ok: sağa git
- Sol ok: sola git
- Boşluk: zıpla
- N: test amacıyla sonraki seviyeye geç

## Kişiselleştirme hedefi

Bu proje Almila'nın fotoğrafları, aile fotoğrafları, çizimleri, sevdiği oyuncaklar ve ses kayıtlarıyla kişiselleştirilecek şekilde tasarlanır.

## Geliştirme yaklaşımı

Scratch `.sb3` dosyası aslında bir ZIP paketidir. İçindeki `project.json` düzenlenebilir. Bu repo, Scratch dosyasının mantığını, JSON notlarını ve geliştirme görevlerini takip etmek için kullanılacaktır.
